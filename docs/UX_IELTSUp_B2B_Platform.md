## 2. 用户流程

### 2.1 BC 运营：完成机构开通

```
[BC 内部登录]
↓
[客户列表] → 点击"创建客户"
↓
[创建客户弹窗] → 填写客户名称、联系人
↓
[客户详情] → 点击"新增合同"
↓
[新增合同页] → 选择产品、配置周期、填写服务截止时间
↓
[系统自动生成订阅] → 提示"订阅已生效"
↓
[订阅详情] → 点击"创建管理员账号"
↓
[创建管理员弹窗] → 填写姓名、邮箱
↓
[系统发送激活邮件] → 提示"激活邮件已发送"
```

### 2.2 机构管理员：发放邀请码

```
[机构首页] → 快捷入口"创建批次"
↓
[邀请码批次 - 步骤 1：产品选择] → 选择 WriteUp/SpeakUp/ScoreUp + 确认余量
↓
[邀请码批次 - 步骤 2：批次设置] → 绑定试卷、填写数量、设置有效期、选择报告释放方式
↓
[邀请码批次 - 步骤 3：消费者信息] → 手动录入 or 上传文件 or 历史批次复用
↓
[预览确认页] → 核对批次信息
↓
[创建成功] → 返回批次列表，新批次高亮显示
```

### 2.3 机构教师：创建试卷

```
[试卷列表] → 点击"创建试卷"
↓
[创建试卷 - 选择产品类型]（WriteUp / SpeakUp）
↓
[左侧题库筛选] + [右侧已选区域]
↓                    ↓
按 Task/Part 筛选   拖拽调整顺序
↓
[结构校验通过] → "保存"按钮亮起
↓
[试卷创建成功] → 返回试卷列表
```

### 2.4 机构用户：查看报告并释放

```
[WriteUp 报告分析页] → 按批次筛选
↓
[报告列表] → 找到目标消费者记录
↓
[报告详情页] → 查看作答结果 + 评分项
↓
（如批次配置为手动释放）
↓
[点击"释放报告"] → 二次确认弹窗
↓
[报告释放成功] → 状态更新为"已释放"
```

## 4. 核心组件体系

### 4.1 状态 Badge 组件

| Badge 类型 | 颜色 | 示例值 |
|-----------|------|-------|
| 生效中 / 进行中 | `#0EA66A` 绿色背景（浅） + 绿色文字 | 生效中 |
| 已到期 / 已过期 | `#9B9B9B` 灰色 | 已到期 |
| 待生效 | `#F5A623` 橙色 | 待生效 |
| 未激活 | `#E8AE00` 黄色 | 未激活 |
| 已停用 / 禁用 | `#D0021B` 红色 | 已停用 |
| 已释放 | `#003F8A` 品牌蓝 | 已释放 |
| 未释放 | `#CFCFCF` 灰边 + 灰字 | 未释放 |

### 4.2 Button 组件规格

| 类型 | 背景色 | 文字颜色 | 高度 | 圆角 |

## 5. 交互说明

### 5.1 全局交互规则

| 场景 | 交互规则 |
|-----|---------|
| 列表搜索 | 点击搜索后刷新列表，重置后恢复初始状态 |
| 高风险操作 | 必须二次确认弹窗，不可单次点击直接执行 |
| 表单提交中 | 主按钮进入 Loading 态，禁止重复点击 |
| 表单校验 | 实时校验（onBlur） + 提交校验（onSubmit） |
| 失败提示 | 提示错误原因 + 操作建议（返回修改 / 重试 / 联系管理员） |
| 弹窗关闭 | 若有未保存编辑，弹出"放弃修改？"二次确认 |
| 页面跳转 | 返回列表页时保留筛选条件 |

### 5.2 客户创建弹窗

1. 点击"创建客户" → 右侧弹出 Drawer 或居中 Modal
2. 客户名称实时重名校验（Debounce 500ms）
3. 提交中：按钮 Loading，表单锁定
4. 成功：弹窗关闭，列表顶部插入新客户，Toast 提示"客户创建成功"
5. 取消时有未输入内容：直接关闭；有已输入内容：二次确认

### 5.3 额度充值弹窗

1. 选择充值档位后，实时展示充值后额度预览
2. 收据编号输入完成（onBlur）后，立即调用接口校验是否重复
3. 重复则即时红色提示，保存按钮禁用
4. 确认提交前展示"充值确认摘要"（金额、档位、截止时间、来源编号）
5. 提交成功：弹窗关闭，订阅详情额度即时刷新

### 5.4 创建试卷 - 题库选题交互

1. 左侧筛选变更后即时刷新题目列表
2. 点击题目行的"加入试卷"：题目加入右侧已选区，左侧该题标记为已选（淡灰底色）
3. 已选题目区支持拖拽排序（`drag handle` 图标拖拽）
4. 题目数量或 Part 结构变动时，右侧实时校验并更新结构检查状态
5. 结构不通过时：保存按钮置灰 + 提示"试卷结构不满足要求"
6. 切换产品类型 → 弹出确认 → 清空已选题目 + 重置筛选

### 5.5 邀请码批次 - 数量输入联动

1. 输入邀请码数量时，实时刷新"本次消耗 / 创建后剩余"
2. 超出可用额度：红色警告文案 + 输入框红色边框 + 保存按钮禁用
3. 数量为空或 0：保存按钮置灰

### 5.6 消费者文件上传

1. 支持拖拽上传和点击上传
2. 上传中：进度条展示
3. 解析完成：展示成功行数 + 失败行列表（逐行错误详情）
4. 允许删除上传文件并重新上传
5. 格式错误时不允许进入下一步，必须修正后重新上传

### 5.7 报告手动释放

1. 释放按钮仅在"未释放"状态下显示
2. 点击后弹出确认弹窗，展示释放目标和影响说明
3. 确认后：按钮消失，状态 Badge 变为"已释放"，记录释放时间和操作人
4. 已释放状态下不展示释放入口

## 7. 响应式说明

### 7.1 桌面端（≥1280px）—— 主要支持目标

- 双栏布局：左侧固定 Sidebar（240px）+ 右侧内容区
- 列表页：全列展示，操作列固定不超过 3 个操作
- 表单：单列或双列布局（2 列时每列 50%）
- 弹窗：最大宽度 `560px`，Drawer 最大宽度 `720px`

### 7.2 平板端（768px–1279px）

- Sidebar 自动收起为图标模式（64px），悬浮展开
- 列表页：隐藏次要列，保留核心列
- 弹窗：85%宽度
- 筛选器：部分筛选项折叠入"更多筛选"

### 7.3 移动端（<768px）

- Sidebar 收起，通过顶部 Hamburger 展开
- 列表简化为卡片列表
- 表单布局改为单列
- 弹窗和 Drawer 改为全屏底部抽屉
- 创建试卷页（双栏选题）：移动端切换为标签页模式（题库 Tab / 已选 Tab）
